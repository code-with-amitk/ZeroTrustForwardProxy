== Local Run Instructions
1. Ensure Go 1.22+ is installed.
2. Install dependencies:
   - `go mod tidy`
3. Configure client/app/browser to use proxy at `localhost:8080`.
4. Import generated root CA (`ca.crt`) into your test client trust store for HTTPS MITM.
5. Scrape metrics at:
   - `http://localhost:9090/metrics`

== Example Policy Behavior
- Rules are first-match-wins.
- Supports wildcard users (`*`) and wildcard domains (`*.example.com`).
- Set `default_action` to `allow` or `block`.

== How to start
=== Server
```
# from repo root — ensure CA exists (ca.crt / ca.key in repo)
docker compose up --build -d
```

=== Client
```
# upload policy for tenant 1
curl -X POST http://127.0.0.1:8090/api/v1/tenants/1/policy \
  -H "Authorization: Bearer changeme" \
  -H "Content-Type: application/json" \
  -d @controlplane/test_policies/policy.json

# run proxy scenarios from host
go run ./cmd/client/ -proxy http://127.0.0.1:8080

# AST dump + then run HTTPS/DLP scenarios through proxy
go run ./cmd/client/ -proxy http://127.0.0.1:8080
```

== Print AST

```bash
# AST only (no HTTP scenarios)
go run ./cmd/client/ -policy-only
```
